# 🛡️ 家庭保险办理

> 启动日期: 2026-04-01
> 目标: 在离职前给全家四口人把保险配齐

## 背景

一直拖着，四个人都还没买。离职后没有公司医保兜底，这件事必须在离职前搞定。

详细方案参考：`areas/finance/insurance.md`

## 优先级

1. **爸爸** — 三高+心脏肥大，能买的窗口在缩小
2. **妈妈** — 52岁，趁身体好赶紧买
3. **星 & 女朋友** — 年轻健康，选择多但别拖

## 待办

### 调研阶段（4月）

- [ ] 找 1-2 个靠谱的保险经纪人/顾问咨询
- [ ] 确认爸爸的健康状况能买哪些产品
- [ ] 天津惠民保了解（投保时间、条件）
- [ ] 爸妈养老保险补交政策咨询（当地社保局）

### 购买阶段（5月）

- [ ] 爸爸：惠民保 + 防癌医疗 + 意外险
- [ ] 妈妈：百万医疗 + 意外险 + 防癌险
- [ ] 星：重疾 + 百万医疗 + 意外 + 定寿
- [ ] 女朋友：重疾 + 百万医疗 + 意外

### 收尾

- [ ] 确认所有保单生效
- [ ] 保单信息整理归档
- [ ] 养老保险档次调整完成

## 预算

全家约 8000-15000/年（详见 `areas/finance/insurance.md`）
